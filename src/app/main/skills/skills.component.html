Habilidades Técnicas

En estos meses en que he podido trabajar en diferentes proyectos, me ha permitido comprender mejor una serie de tecnologías, para el buen desarrollo de software siguiendo la arquitectura modelo vista controlador (MVC).

Angular

Framework para aplicaciones web desarrollado en TypeScript, de código abierto.

Bootstrap

Es una biblioteca multiplataforma o conjunto de herramientas de código abierto para diseño de sitios y aplicaciones web

TypeScript

lenguaje de programación libre y de código abierto desarrollado y mantenido por Microsoft, es un superconjunto de JavaScript.

Diseño web adaptable

Es una filosofía de diseño y desarrollo cuyo objetivo es adaptar la apariencia de las páginas web al dispositivo que se esté utilizando para visitarlas.

API RESTful

Es una interfaz que dos sistemas de computación utilizan para intercambiar información de manera segura a través de Internet.

MCV

Patrón de diseño de software que separa los modelos de presentación, lógica de negocio y manipulación de datos.
